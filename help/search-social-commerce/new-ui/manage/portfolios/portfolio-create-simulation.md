# Ejecutar una simulación personalizada desde la vista [!UICONTROL Portfolios]

*característica de Beta*

Puede crear una simulación personalizada para un portafolio [optimizado o activo](/help/search-social-commerce/new-ui/manage/portfolios/portfolio-about.md) desde la vista [!UICONTROL Portfolios].

>[!NOTE]
>
>También puede [crear o volver a ejecutar una simulación personalizada desde la vista [!UICONTROL Simulations]](/help/search-social-commerce/new-ui/plan/simulations/simulation-create.md).

1. En el menú principal, haga clic en **[!UICONTROL Manage]>[!UICONTROL Portfolios]**.

1. Realice una de las acciones siguientes:

   * Mantenga el cursor sobre la fila del portafolio. Junto al nombre del portafolio, haga clic en **[!UICONTROL ...]** > **[!UICONTROL Run Simulation]**.

   * Seleccione la casilla de verificación situada junto al portafolio. En la barra de herramientas de acciones masivas, haga clic en **[!UICONTROL Run Simulation]**.
