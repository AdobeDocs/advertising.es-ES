# Columnas de informes para informes básicos y avanzados

| Columna | Descripción |
| ---- | ---- |
| \[Métricas personalizadas (derivadas) específicas del anunciante\] | El valor de [métrica personalizada que creó](/help/search-social-commerce/common-tasks/custom-metrics/custom-metric-about.md) que se calcula a partir de métricas existentes. |
| \[Clasificaciones de etiquetas específicas del anunciante\] | Cualquier clasificación de etiquetas aplicada actualmente a la entidad, en el nivel de entidad. Las clasificaciones de varias etiquetas se separan con comas (,). |
| \[Métricas de conversión específicas del anunciante\] | Número de conversiones de una métrica de conversión o una métrica de participación del sitio especificadas. |
| \[Conversiones rastreadas en Google\] | Consulte la entrada para &quot;GGL\*, GGL_CT\* y GGL_XD_CT\*&quot;. |
| [!UICONTROL 7-Day Click Accuracy] | ([!UICONTROL Portfolio Report]) La precisión promedio de la previsión de clics para los siete días anteriores, sin incluir el día actual (y no para el intervalo de fechas especificado del informe), expresada como porcentaje. |
| [!UICONTROL 7-Day Cost Accuracy] | ([!UICONTROL Portfolio Report]) La precisión promedio de la previsión de costos de los siete días anteriores, sin incluir el día actual (y no para el intervalo de fechas especificado en el informe), expresada como porcentaje. |
| [!UICONTROL 7-Day Revenue Accuracy] | ([!UICONTROL Portfolio Report]) La precisión promedio de la previsión de ingresos de los siete días anteriores, sin incluir el día actual (y no para el intervalo de fechas especificado en el informe), expresada como porcentaje. |
| [!UICONTROL Account] | El nombre de la cuenta. |

Los datos recopilados mientras la cuenta estaba habilitada se siguen almacenando, pero todas las vistas de administración de campañas y todos los informes que cree en el futuro no incluyen datos del período de tiempo en el que la cuenta está deshabilitada.<br><br>En las redes de anuncios no sincronizadas, las características aplicables, como la optimización o la generación de seguimiento, no están disponibles.</li></ul> |

| [!UICONTROL Ad Group Type] | El tipo de grupo de anuncios, como <i>[!UICONTROL Audience]</i> (solo para campañas de audiencia), <i>[!UICONTROL Demand Gen]</i> (solo para campañas de generación de demanda), <i>[!UICONTROL Display]</i> (solo para campañas de visualización), <i>[!UICONTROL Search Dynamic]</i> (solo para anuncios dinámicos de búsqueda), <i>[!UICONTROL Search Standard]</i> (solo para anuncios de búsqueda interactivos y anuncios de texto expandidos existentes), <i>[!UICONTROL Shopping Showcase]</i>, <i>[!UICONTROL Shopping Product]</i> (solo para campañas de compra estándar) o <i>[!UICONTROL Shopping Smart]</i> (para campañas de compra inteligente). | [!UICONTROL AD Name] | El nombre del grupo de anuncios; el mismo valor que [!UICONTROL Ad Group]. |
| [!UICONTROL Ad Recall Lift] | ([!DNL Meta] campañas solamente) El número estimado de personas que recuerdan su anuncio en un plazo de dos días. |
| [!UICONTROL Ad Recall Rate] | ([!DNL Meta] campañas solamente) El número estimado de personas que recuerdan su anuncio en un plazo de dos días dividido por el número de personas que ha llegado a un nivel, como porcentaje. |

| [!UICONTROL AD Strength] | ([!DNL Google Ads] anuncios de búsqueda adaptables) La eficacia del anuncio: <i>[!UICONTROL average]</i>, <i>[!UICONTROL excellent]</i>, <i>[!UICONTROL good]</i>, <i>[!UICONTROL no_ads]</i>, <i>[!UICONTROL pending]</i>, <i>[!UICONTROL poor]</i>, <i>[!UICONTROL unknown]</i> o <i>[!UICONTROL unspecified]</i>. |
| [!UICONTROL Adgroup MBA] | ([!DNL Google Ads], [!DNL Microsoft Advertising] y [!DNL Yahoo! Japan Ads] campañas) El ajuste de oferta móvil actual de nivel de grupo de anuncios, que determina cómo se ajustan las ofertas cuando el anuncio se muestra en un dispositivo móvil. |
| [!UICONTROL AI Max Bundling Required] | (Campañas dirigidas únicamente a la red de búsqueda; campañas con la función AI Max habilitada; solo lectura) Si se requiere el agrupamiento: *[!UICONTROL REQUIRED]*, *[!UICONTROL NOT_REQUIRED]*, *[!UICONTROL UNSPECIFIED]* o nulo. |
| [!UICONTROL AI Max Enabled] | Si la característica [[!UICONTROL AI Max] &#x200B;](https://support.google.com/google-ads/answer/15910366) está habilitada: [!UICONTROL true]*, *[!UICONTROL false]* o es nula. |
| [!UICONTROL AI Max Search Term Matching] | (Campañas dirigidas a la red de búsqueda para las que la característica [AI Max](https://support.google.com/google-ads/answer/15910366) y la característica de coincidencia de términos de búsqueda en el nivel de campaña están habilitadas; solo lectura) Indica si la coincidencia de términos de búsqueda en el nivel de grupo de anuncios está habilitada: *[!UICONTROL true]*, *[!UICONTROL false]* o nula. |
