# [!UICONTROL Label Value Report]

[!UICONTROL Label Value Report] incluye datos de costo, clics y (opcionalmente) conversión por valor de clasificación de etiqueta agregado en portafolios, redes de anuncios, cuentas, campañas o grupos de anuncios. De forma predeterminada, los datos incluyen una fila para cada valor aplicable de palabras clave, anuncios y ubicaciones que recibieron impresiones para cada unidad de tiempo en el intervalo de fechas especificado. Las filas están en orden ascendente, primero por la fecha de inicio de la unidad de tiempo, después por el coste y, por último, por el valor de la etiqueta, de forma predeterminada. También puede ver el número de cada tipo de entidad al que se asigna el valor de la etiqueta.

Puede ver los datos de los 36 meses anteriores.

>[!NOTE]
>
>Cualquier cambio que realice en las clasificaciones de etiquetas y en los valores de etiquetas secundarias de una entidad será visible en aproximadamente una hora.
