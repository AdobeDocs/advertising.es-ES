# [!UICONTROL MSA Ad Extension by Keyword Report]

*[!DNL Microsoft Advertising]solo cuentas*

Este informe muestra las impresiones, los clics, los gastos y el coste promedio por clic de cada extensión de anuncio y el tipo de clic por palabra clave. Utilice este informe para comparar el rendimiento de las extensiones de publicidad de cada palabra clave.

Cuando se ejecuta un informe solamente con columnas de métricas y sin dimensiones, el informe generado contiene dos filas duplicadas: una con datos agregados y otra con todas las métricas.<!-- all metrics? -->

Puede ver los datos de los últimos 180 días. Search, Social y Commerce extrae los datos de [!DNL Microsoft Advertising] en tiempo real y puede tardar varios minutos en generar informes con un volumen de datos mayor.

>[!NOTE]
>
>Cuando el informe incluye un gran volumen de datos, es posible que se produzca un error. Si esto sucede, ejecute el informe durante un intervalo de tiempo menor.
