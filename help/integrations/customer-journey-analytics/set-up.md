# Configuración de la recopilación de datos, la transferencia de datos y la creación de informes

Se requieren las siguientes tareas para ver los datos de Advertising Cloud en Customer Journey Analytics.

<!--
The following procedures explain the steps required to view Advertising Cloud data in Customer Journey Analytics. The procedures include the tasks and settings required for the integration but do not explain all features available for the workflows; see the linked resources for full information.
-->

1. (Analista web de su organización; opcional) [Recopilar datos históricos para los ID de AMO e ID de EF](/help/integrations/analytics/rvars-to-evars.md){target="_blank"}.

   Este paso solo es aplicable a los anunciantes con [!DNL Analytics for Advertising].

1. (El administrador del sitio de su organización para Adobe Experience Platform) Configure la recopilación de datos en Experience Platform e implemente las etiquetas de seguimiento de conversiones.

1. (Administrador del sitio de su organización para Customer Journey Analytics) Cree una conexión a los conjuntos de datos de Experience Platform en Customer Journey Analytics.

1. (Analista web de su organización) Configure las vistas de datos en Customer Journey Analytics.

1. (Analista web de su organización) Configure informes y visualizaciones en Customer Journey Analytics Workspace.
