# Novedades [!DNL Creative] de 2.0

Las siguientes funciones son nuevas o se han modificado recientemente.

| Fecha | Característica | Descripción | Para más información |
| ---- | ------- | ----------- | -------------------- |
| 21 enero 2026 | Creative Library, Adobe Systems GenStudio para Performance Marketing | Los anunciantes con [Adobe Systems GenStudio for Performance Marketing](https://experienceleague.adobe.com/en/docs/genstudio-for-performance-marketing/user-guide/home) ahora pueden importar experiencias anuncio en pantalla de GenStudio a su bibliotecas creativa. <!-- Each GenStudio experience is saved as an HTML5 creative.--> Puede incluir los elementos creativos importados en sus experiencias de Creative publicitarias. | Consulte &quot;[Acerca de los bibliotecas creativa](/help/creative/creative-libraries/creative-libraries-about.md)&quot; y &quot;[T añadir elementos creativos estándar a una biblioteca creativa](/help/creative/creative-libraries/creative-add-standard.md)&quot;. |
|  | Elementos creativos estándar | Las opciones de menú para crear imágenes y anuncios HTML5 ahora se combinan en una sola opción &quot;[!UICONTROL Standard Display]&quot;, que también puede usar para importar experiencias de anuncio en pantalla desde GenStudio. | Consulte &quot;[Acerca de los bibliotecas creativa](/help/creative/creative-libraries/creative-libraries-about.md)&quot; y &quot;[T añadir elementos creativos estándar a una biblioteca creativa](/help/creative/creative-libraries/creative-add-standard.md)&quot;. |
| 17 diciembre 2025 | Alertas | Un [!UICONTROL Pulse Panel] icono en la [!UICONTROL Experiences] vista indica si hay alertas disponibles para sus experiencias. Puede abrir el panel para vista cada alerta y, opcionalmente, abrir el experiencia o creativa aplicable para que pueda editarlo según sea necesario para resolver el problema. Opcionalmente, también puede descartar cualquier alerta.<br><br>Para los anunciantes con DSP publicitarias, las alertas sobre DSP ubicaciones creadas a partir de experiencias están disponibles en el nivel campaña dentro de [!DNL Creative] DSP. | Consulte &quot;[Ver alertas](/help/creative/reports/alerts-view.md)&quot;. |
| 26 noviembre 2025 | Costo sistema de informes de Creative publicitaria | (Anunciantes con Creative publicitaria y DSP publicitaria solamente; función beta) El nuevo [!UICONTROL All-in Cost] informe está fuera de versión beta y está disponible de forma general. | Consulte &quot;[Acerca de los informes personalizados](/help/creative/reports/reports-about.md)&quot; y &quot;[Administrar informes personalizados](/help/creative/reports/report-manage.md)&quot;. |
| 17 noviembre 2025 | Costo sistema de informes de Creative publicitaria | (Anunciantes con Creative publicitaria y DSP publicitaria solamente; función beta) El nuevo [!UICONTROL All-in Cost BETA] informe muestra cuánto se gastó a servicio de publicidad para Adobe Systems Creative. Las métricas de gasto del informe incluyen [!UICONTROL Billable Creative Spend], [!UICONTROL Non Billable Creative Spend], [!UICONTROL Total Creative Spend], [!UICONTROL Total Creative CPM], y .[!UICONTROL Estimated Tax on Creative] Puede vista datos de creativa, atributo, destino y de otro tipo en los niveles de campaña, paquete, ubicación y anuncios. | Consulte &quot;[Acerca de los informes personalizados](/help/creative/reports/reports-about.md)&quot; y &quot;[Administrar informes personalizados](/help/creative/reports/report-manage.md)&quot;. |
| 24 septiembre 2025 | Creatividades dinámicas | Ahora dispone de un flujo de trabajo de autoservicio más sencillo para generar anuncios dinámicos utilizando datos de un catálogo de variaciones anuncios que pasan por un plantilla de anuncios. | Consulte &quot;[Flujos de trabajo para publicidad](/help/creative/introduction/workflow-dynamic-ads.md) dinámica&quot;. |
| 3 septiembre 2025 | Creative optimización y programación de [!DNL experiences] | La opción de menú o botón para configurar la optimización y programación de creativa ahora se llama &quot;[!UICONTROL Creative Optimization]&quot; en lugar de &quot;[!UICONTROL Edit Schedules]&quot; o &quot;[!UICONTROL Add Schedules].&quot; | Consulte &quot;[Personalizar la optimización creativa y la programación para un experiencia](/help/creative/experiences/experience-optimization-scheduling-targeting.md) específico&quot; y &quot;[Personalizar la optimización y programación de creativa para un experiencia sin direccionamiento](/help/creative/experiences/experience-optimization-scheduling-no-targeting.md)&quot;. |
|  | Rotación de anuncios y programación para [!DNL experiences] | Un nuevo tipo de rotación de creativa, &quot;[!UICONTROL Sequencing],&quot; rota los paquetes de creativa asociados en un orden especificado (con el paquete 1 servido primero, el paquete 2 servido en segundo lugar, etc.), con un número total especificado de impresiones en cada paquete Secuencia. Puede configurar las paquete finales en el Secuencia a a\) para que se muestren indefinidamente o b\) volver al primer paquete. | Consulte &quot;[Personalizar la optimización creativa y la programación para un experiencia](/help/creative/experiences/experience-optimization-scheduling-targeting.md) específico&quot; y &quot;[Personalizar la optimización y programación de creativa para un experiencia sin direccionamiento](/help/creative/experiences/experience-optimization-scheduling-no-targeting.md)&quot;. |
| 4 agosto 2025 | Objetivos de audiencia para anuncios [!DNL experiences] | Una lógica más compleja para audiencia inclusión y exclusión (la misma lógica disponible en Advertising DSP) está disponible dentro de audiencia configuración destino. Al configurar destinos de audiencia para un experiencia anuncios, ahora puede configurar varios grupos de segmento y especificar si desea incluir o excluir usuarios según la lógica.<br><br>Puede ver el número total de miembros de audiencia para cada segmento y el número de miembros activos y totales de audiencia para cada segmento grupo.<br><br>Con esta opción, la opción existente para &quot;[!UICONTROL Split targets to create nodes]&quot; ya no está disponible.<br><br>Si ya contaba con nodos audiencia que destino varios segmentos, ahora se segmentan mediante la opción &quot;[!UICONTROL Include Any]&quot; y el usuario debe pertenecer a uno solo de los segmentos para poder optar a un impresión. Anteriormente, el usuario debía pertenecer a todos los segmentos. | Consulte &quot;[añadir un destino nodo al nivel](/help/creative/experiences/experience-target-node-add-final.md) final&quot;, &quot;[Insertar un nodo destino entre nodos](/help/creative/experiences/experience-target-node-add-inner.md)&quot; y &quot;[añadir un destino del mismo nivel nodo](/help/creative/experiences/experience-target-node-add-sibling.md)&quot;. |
| 10 julio 2025 | Vídeo creativos | Ahora hay compatibilidad disponible para creativos de vídeo propios y para paquetes y experiencias específicos de vídeo:<ul><li>Ahora puede cargar creatividades de vídeo de origen y agregarlas a paquetes específicos de vídeo. En paquete configuración, las opciones &quot;[!UICONTROL Bundle Type]&quot; ahora incluyen [!UICONTROL Standard Display], [!UICONTROL Dynamic Display], y [!UICONTROL Standard Video].</li><li>Puede crear experiencias de anuncios específicas de vídeo con los paquetes de vídeo. La configuración de experiencia de anuncios ahora incluye un ajuste &quot;[!UICONTROL Ad Type]&quot;, con [!UICONTROL Standard Display], [!UICONTROL Dynamic Display]y [!UICONTROL Video] opciones. Puede optimizar los anuncios de vídeo según la tasa de pulsaciones, la tasa de finalización o un objetivo personalizado.</li><li>Las etiquetas para anuncios de vídeo experiencia se definen por la duración y la velocidad de bits del vídeo, en lugar de por anuncios tamaño.</li><li>Vídeo anuncios se transcodifican automáticamente a Adobe Systems codificación de DSP publicitaria para que pueda previsualización ellos. Opcionalmente, puede aplicar la transcodificación para otros DSP a cualquier anuncios experiencia etiqueta dentro de [!UICONTROL Tag Manager].</li></ul> | Consulte &quot;[Acerca de su creativa bibliotecas](/help/creative/creative-libraries/creative-libraries-about.md)&quot;, &quot;[Administrar paquetes](/help/creative/creative-libraries/bundle-manage.md) de creativa&quot;, &quot;[Configuración](/help/creative/experiences/experience-settings-targeting.md) de experiencia objetivo&quot;, &quot;[Configuración](/help/creative/experiences/experience-settings-no-targeting.md) de experiencia no segmentada y &quot;[Personalizar las opciones de transcodificación para un anuncios de vídeo experiencia etiqueta](/help/creative/experiences/experience-tag-video-transcoding.md)&quot;. |
| 21 mayo 2025 | [!UICONTROL Creative Libraries] | Ahora puede agregar imágenes de sus Adobe Experience Manager recurso biblioteca a sus [!UICONTROL Creative Libraries] para poder usarlas dentro de anuncios experiencias. | Consulte &quot;[Configuración del acceso a Adobe Experience Manager activos](/help/creative/creative-libraries/aem-assets-configure.md) de imagen&quot; y &quot;[T añadir elementos creativos estándar a una biblioteca creativa](/help/creative/creative-libraries/creative-add-standard.md)&quot;. |
| 10 febrero 2025 | [!UICONTROL Creative Libraries] | Anteriormente, tenías una creativa biblioteca. Ahora puede crear varias bibliotecas para cada anunciante. | Consulte &quot;[Acerca de su creativa bibliotecas](/help/creative/creative-libraries/creative-libraries-about.md)&quot;. |
| | [!UICONTROL Creative Libraries] > [!UICONTROL Creatives] | El [!UICONTROL Creatives] vista incluye fichas para [!UICONTROL Standard Ads] y [!UICONTROL Dynamic Ads].<ul><li>El **[!UICONTROL Standard Ads]pestaña** le permite cargar y administrar imágenes, HTML5, HTML5 flexible y creatividades terceros.</li><li>El **[!UICONTROL Dynamic Ads]** pestaña permite administrar anuncios generados dinámicamente que se crean a partir de archivos de fuente cargados utilizando plantillas de anuncios definidas; anteriormente, los anuncios dinámicos se generaban en [!DNL Adobe Advertising Dynamic Creative Optimization (DCO)].<br><br> Actualmente puede previsualización, duplicado y eliminar publicidad dinámica. También puede adjuntar publicidad dinámica a paquetes de creativa para experiencias de anuncios objetivo o a etiquetas de anuncios para experiencias no segmentadas. Solo los usuarios administradores pueden generar anuncios de forma dinámica.</li></ul> | Consulte &quot;[Acerca de su creativa bibliotecas](/help/creative/creative-libraries/creative-libraries-about.md)&quot;. |
| | [!UICONTROL Creative Libraries] > [!UICONTROL Bundles] | Agrupe varios elementos creativos en una *paquete* para agregarlos fácilmente a una experiencia. Puede crear paquetes de anuncios estándar y adjuntarles elementos creativos estándar. Del mismo modo, puede crear paquetes de anuncios dinámicos y adjuntar creatividades dinámicas a ellos. | Consulte &quot;[Administrar paquetes de Creative](/help/creative/creative-libraries/bundle-manage.md)&quot;. |
| | [!UICONTROL Experiences] | En la nueva configuración de experiencia de anuncios, ahora se especifica si el experiencia utiliza o no el árbol de decisiones direccionamiento, y no se puede cambiar la configuración una vez que haya guardado el experiencia. Los flujos de trabajo para las experiencias con direccionamiento de árbol de decisión y las experiencias sin direccionamiento de árbol de decisión son diferentes. | Consulte &quot;[Crear una experiencia con direccionamiento](/help/creative/experiences/experience-create-targeting.md)&quot; y &quot;[Crear una experiencia sin direccionamiento](/help/creative/experiences/experience-create-no-targeting.md)&quot;. |
| | [!UICONTROL Experiences] | Ahora solo puede crear experiencias dirigidas con paquetes de creativa de un solo creativa biblioteca, no con creatividades individuales. Puede seguir adjuntando elementos creativos individuales de una sola biblioteca a experiencias no segmentadas sin necesidad de direccionamiento de árbol de decisiones.<br><br>Debido a los cambios estructurales, sus experiencias heredadas quedarán obsoletas a finales de este año. | Clientes de autoservicio: reconstruya sus experiencias en la nueva interfaz de usuario. Consulte &quot;[Crear una experiencia con direccionamiento](/help/creative/experiences/experience-create-targeting.md)&quot;.<br><br>Clientes de servicios administrados: el equipo de cuentas de Adobe Systems reconstruirá sus experiencias en la nueva interfaz de usuario. |

| | Experiencias DCO | Las experiencias de DCO heredadas quedarán obsoletas a finales de este año.
